# 🔄 API URL Индикатор Обновления

## Что добавилось?

Теперь в вкладке **⚙️ Настройки** рядом с полем `API URL` есть индикатор который показывает статус:

### 📊 Индикаторы:

| Значок | Цвет | Смысл |
|--------|------|-------|
| **✅** | 🟢 Зелёный | API URL сохранён и используется |
| **⚠️** | 🟠 Оранжевый | API URL был изменён, нужно нажать "💾 Сохранить" |

## 🔄 Как это работает?

1. **При загрузке приложения:**
   - Индикатор показывает ✅ зелёный (текущий URL сохранён)

2. **Когда вы меняете URL в поле:**
   - Индикатор меняется на ⚠️ оранжевый
   - Это означает что изменения ещё не сохранены

3. **Когда вы нажимаете "💾 Сохранить":**
   - Индикатор меняется на ✅ зелёный
   - Новый URL теперь активен
   - Приложение будет отправлять донаты на этот новый адрес

## 💡 Пример использования:

```
1. Открываешь вкладку ⚙️ Настройки
2. Видишь API URL с ✅ (всё хорошо)
3. Меняешь ссылку на: https://webhook-test.com/new-id
4. Видишь индикатор изменился на ⚠️
5. Нажимаешь "💾 Сохранить"
6. Индикатор становится ✅
7. Теперь новый вебхук получит донаты!
```

## 🎯 Преимущества:

✅ Сразу видно изменился ли URL  
✅ Можно отменить изменения (отредактировать обратно)  
✅ Понятный визуальный сигнал о статусе  
✅ Не забудешь нажать "Сохранить"
