# 📢 ПОСЛЕДНИЕ ОБНОВЛЕНИЯ (26.10.2025)

## 🚀 Новое

### 1. ✅ Исправлена ошибка "Event loop is closed"

**До:** При отправке данных на webhook возникала ошибка и данные не отправлялись
```
❌ Ошибка отправки на webhook: Event loop is closed
RuntimeError: Event loop is closed
```

**После:** Event loop работает корректно, данные отправляются на webhook без ошибок

**Технические подробности:**
- Изменён способ обработки асинхронных операций
- Теперь используется один persistent event loop вместо создания нового loop для каждого уведомления
- Event loop закрывается только при завершении приложения

---

### 2. 🎯 Добавлен индикатор статуса API URL

Теперь видно обновился ли вебхук или нет!

```
⚙️ Настройки → 🔌 Подключение
┌─────────────────────────────────────────────┐
│ API URL: [https://webhook.com/......]  ✅   │
│                                    ↑        │
│                         Индикатор статуса   │
└─────────────────────────────────────────────┘
```

**Состояния:**
- ✅ **Зелёный** = URL сохранён и используется (Webhook получит данные)
- ⚠️ **Оранжевый** = URL был изменён, нужно нажать "Сохранить"

**Как использовать:**
1. Меняешь URL webhook → индикатор ⚠️
2. Нажимаешь "💾 Сохранить" → индикатор ✅
3. Отправляешь донат → приложение использует новый URL
