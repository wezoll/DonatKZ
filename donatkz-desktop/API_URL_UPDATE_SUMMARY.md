# 🔧 Исправления (26.10.2025)

## 1. ✅ Исправлен Event Loop Error при отправке на webhook

**Проблема:** При отправке данных на webhook возникала ошибка "Event loop is closed"

**Причина:** Для каждого уведомления создавался новый event loop через `asyncio.run()`, который закрывался после выполнения. Потом когда нужно было отправить на webhook, loop уже был закрыт.

**Решение:**
- Создан **один persistent event loop** в `DonationPipeline.__init__()`
- Использован `self.loop.run_until_complete()` вместо `asyncio.run()`
- Event loop закрывается только при завершении приложения

**Файлы изменены:**
- `src/core/donation_pipeline.py` - добавлены `self.loop` и его управление

## 2. 🎯 Добавлен визуальный индикатор обновления API URL

**Функция:** Теперь в вкладке "⚙️ Настройки" рядом с полем API URL есть индикатор статуса

**Индикаторы:**
- ✅ **Зелёный** = API URL сохранён и используется
- ⚠️ **Оранжевый** = API URL был изменён, нужно сохранить

**Как работает:**
1. При загрузке - индикатор ✅ зелёный
2. Когда вы меняете URL - индикатор ⚠️ оранжевый
3. Когда вы сохраняете - индикатор ✅ зелёный
4. При наведении на индикатор - появляется подсказка

**Файлы изменены:**
- `src/ui/settings_tab.py` - добавлены:
  - `self.api_update_indicator` - виджет с индикатором
  - `self.current_api_url` - сохранённый URL для сравнения
  - `_on_api_url_changed()` - обработчик изменений
  - Обновлён `load_settings()` и `save_settings()`
- `src/ui/widgets.py` - добавлен класс `SimpleTooltip` для подсказок
