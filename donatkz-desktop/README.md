# DonatKZ Desktop v0.1.0

🎁 **Современное приложение для отслеживания донатов из Kaspi** через Windows Phone Link в реальном времени.

## ✨ Возможности

- ✅ **Перехват уведомлений Kaspi** - получает доны прямо из Phone Link
- ✅ **Гибкая фильтрация** - автоматически определяет пополнения и переводы
- ✅ **Дедупликация** - исключает дубликаты даже при перезапусках
- ✅ **Правильное время** - извлекает время из Phone Link (не системное время)
- ✅ **Dashboard статистика** - показывает сумму и среднее значение
- ✅ **Простой индикатор** - 🟢 Phone Link работает / 🔴 не работает
- ✅ **Масштабируемость** - легко обрабатывает 100+ донатов

## 📋 Требования

- **Windows 10/11**
- **Python 3.10+**
- **Phone Link** установлен и настроен
- **Android телефон** с Kaspi подключен к Phone Link

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
python src/main.py
```

### 3. Первый запуск

- Приложение запросит разрешение на доступ к уведомлениям Windows
- Убедитесь что Phone Link активен и подключен к телефону
- Статус покажет 🟢 **"Работает"** когда Phone Link готов

## 🔧 Как это работает

### 1. Windows Notification Listener

```
Phone Link → Windows Notifications → winsdk API → Listener
                                          ↓
                                   Парсинг текста
                                          ↓
                                    Фильтрация Kaspi
                                          ↓
                                   Дедупликация
                                          ↓
                                    GUI обновление
```

### 2. Парсинг донатов

Приложение поддерживает форматы:
- ✅ `Пополнение: 100 ₸ Аспандияр Т.: Сообщение`
- ✅ `Перевод: 500 ₸ от Айгерим К.`
- ✅ `100 ₸ от Максим С. - Спасибо за стрим`
- ✅ И многие другие...

### 3. Дедупликация

- Генерирует **стабильный хеш** на основе содержимого (app_id + текст + время)
- Сохраняет кэш между перезапусками слушателя
- Окно: **60 секунд** - считает дубликатом за 60 сек
- Автоматически очищает кэш при достижении 200 элементов

### 4. Фильтрация по времени

- Игнорирует уведомления **старше 5 минут**
- Предотвращает обработку старых донатов при перезапусках

## 📊 Статистика в Dashboard

- **Донатов** - количество уникальных донатов сегодня
- **Сумма** - общая сумма всех донатов
- **Средняя** - среднее значение одного доната

## 🔐 Разрешения

При первом запуске приложение запросит разрешение на:
- **Доступ к уведомлениям** - для перехвата Kaspi донатов
- **Доступ к трею** - для иконки в системном трее

Если отказать:
- Приложение покажет 🔴 **"Не работает"**
- Никакие доны не будут перехватываться

## 🐛 Решение проблем

### Phone Link показывает 🔴 "Не работает"

**Решение:**
1. Откройте **Параметры → Конфиденциальность → Уведомления**
2. Включите уведомления для приложения
3. Перезагрузите приложение

### Доны не перехватываются

**Проверьте:**
1. ✅ Phone Link установлен и активен
2. ✅ Телефон подключен к Phone Link
3. ✅ Синхронизация уведомлений включена
4. ✅ Kaspi приложение на телефоне активно

### Дубликаты в таблице

**Это нормально если:**
- Вы только что перезагрузили приложение (очищается кэш)
- Уведомление старше 5 минут (переобработка)

**После переустановки слушателя - кэш персистируется**, дубликатов быть не должно!
